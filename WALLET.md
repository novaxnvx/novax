# NOVAX WALLET SYSTEM

## 1. ¿Qué es una wallet?
Una wallet NOVAX es una identidad digital única.
No es un número inventado ni un correo.

## 2. Cómo se crea una wallet
La wallet se genera localmente mediante:
- Una clave privada secreta
- Una clave pública derivada

La clave pública es la wallet visible.
La clave privada nunca se comparte.

## 3. Existencia real
Una wallet existe solo si:
- Fue generada correctamente
- Está registrada en el sistema
- Tiene historial válido

Si una wallet no cumple esto, el sistema la rechaza.

## 4. No correos, no terceros
NOVAX no depende de:
- Correos públicos
- Plataformas externas
- Servicios de custodia

El control es del usuario.

## 5. Pérdida
Si el usuario pierde su clave privada:
- La wallet no puede recuperarse
- Los fondos quedan inaccesibles

## 6. Uso
Las wallets sirven para:
- Recibir NVX
- Enviar NVX
- Participar en el sistema

Nada más.
