# NOVAX – MINING

## 1. Qué es la creación de bloques
La red NOVAX genera nuevas monedas
mediante un proceso computacional
basado en validación de bloques.

Cada bloque válido agrega registros
y libera nuevas unidades NOVAX.

## 2. Participación
Cualquier persona puede participar
ejecutando el software de la red.

No se requiere registro ni permiso.

## 3. Recompensa por bloque
Cada bloque válido genera:

- 98% para el validador del bloque
- 2% para el creador del proyecto

La comisión del 2% se utiliza
para mantenimiento, desarrollo
y continuidad del sistema.

## 4. Emisión total
El suministro máximo de NOVAX es:

21,000,000 NVX

No se podrán crear más monedas
una vez alcanzado este límite.

## 5. Dificultad
La dificultad de creación de bloques
aumenta progresivamente con el tiempo.

Esto garantiza:
- Ritmo controlado
- Seguridad
- Estabilidad del sistema

## 6. Ajuste
La dificultad se ajusta automáticamente
según la actividad de la red.

## 7. Control
Las reglas de emisión son públicas
y no pueden ser modificadas
sin consenso documentado.

## 8. Estado actual
El sistema se encuentra en fase inicial
y sujeto a pruebas controladas.
